Front-end Demonstration

Concept 1 - The ML Oracle

  • One machine learning model
  • Reports a score with no explanation
  • Reports the confidence of the rating

Concept 2 - The Explainable ML Oracle

  • One machine learning model
  • Explain each feature contribution

Concept 3 - The Multi-Model ML Assistant

  • A tool that provides risk classifications for multiple Risk Factors, RFi
  • Each Risk Factor feeds into a machine learning model
  • The Risk Factor models produce a Risk Classification, RCRFi
  • The overall change request Risk Classification is, RCchange request = max(RCRF0, ..., RCRFn)
