The head 1 to head 3 titles are used for display risk from high risk to low risk.
The background-color is #0052CC a dainty blue color same as Atlassian style.
The same style allows the user to use both software in one page without feeling cut off.
The white head is clear and easy for the user to differentiate when the background is blue.

Normal text and head use the same font-family
